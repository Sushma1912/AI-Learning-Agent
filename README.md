# 🧠 AI Learning Agent (Multi-Agent System)

## Overview
AI Learning Agent is a multi-agent intelligent system designed to help users generate personalized study plans, evaluate learning progress, and track performance using AI-based reasoning.

The system simulates real-world learning workflows using multiple specialized agents working together.

---

## ⚙️ Features

- 📘 Personalized Study Plan Generator
- 🧪 AI-based Assessment System
- 📊 Learning Progress Tracker
- 🧠 Multi-Agent Architecture
- 🔄 Adaptive Learning Recommendations

---

## 🏗️ Architecture

The system consists of 3 core agents:

### 1. 📘 Planner Agent
- Creates structured study plans
- Breaks syllabus into daily learning tasks

### 2. 🧪 Assessment Agent
- Evaluates learner performance
- Generates scores and feedback

### 3. 📊 Tracker Agent
- Tracks progress over time
- Shows improvement insights
